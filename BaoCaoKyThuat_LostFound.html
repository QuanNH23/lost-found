4.2. VỀ COOKIES

Câu hỏi: "Giỏ hàng dùng cookies ở đâu..."

Dự án không dùng Cookie vì đây là ứng dụng tìm đồ thất lạc, không phải thương mại điện tử. Toàn bộ state quản lý bằng HttpSession. Cookie duy nhất là JSESSIONID do Tomcat tự tạo.

4.3. MODULE COMMENT & REPLY

Xem bình luận (doGet) — Xây dựng Comment Tree

itemDetailController.java — forwardDetail() dòng 97-123

// Lấy tất cả message, lọc ra Comment
List<Message> comments = new ArrayList<>();
for (Message msg : messageDAO.getMessagesByItemId(itemId)) {
    if ("Comment".equalsIgnoreCase(msg.getTitle())) comments.add(msg);
}

// Xây dựng CÂY BÌNH LUẬN (parent-child tree)
List<CommentNode> rootNodes = new ArrayList<>();
Map<Integer, CommentNode> nodeMap = new HashMap<>();
for (Message c : comments) nodeMap.put(c.getMessageId(), new CommentNode(c));
for (Message c : comments) {
    CommentNode node = nodeMap.get(c.getMessageId());
    if (c.getParentId() == null || c.getParentId() == 0) {
        rootNodes.add(node);          // Comment gốc
    } else {
        nodeMap.get(c.getParentId()).getReplies().add(node); // Reply → gắn vào cha
    }
}
request.setAttribute("commentTree", rootNodes);
